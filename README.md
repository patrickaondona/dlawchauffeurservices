# Dlaw Chauffeur Services

Premium chauffeur and mobility website focused on Abuja, Nigeria. The site highlights a luxury fleet, booking flows, and supporting services beyond rides.

**What It Includes**
- Homepage with hero, fleet preview, services overview, and “why choose us” highlights.
- Fleet catalog with individual car listings.
- Booking modal that sends requests via WhatsApp and email.
- Contact support modal and footer contact section.
- Privacy Policy and Terms of Service pages.

**Key Routes**
- `/` homepage
- `/cars` full fleet
- `/cars/[id]` car detail pages
- `/privacy-policy`
- `/terms-of-service`
- `/login`

**Tech Stack**
- Next.js App Router
- React 19
- Tailwind CSS v4
- TypeScript

**Run Locally**
```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
